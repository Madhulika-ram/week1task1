## Personal_intro
## 1. Project Overview:
This project is a simple python program implemented in a jupyter notebook that collects a users personal introduction and prints it in aclean , formatted way.
The notebook contains:
  - Name
  - Age
  - Hobbies
  - Skills
Used to understand:
  - Usage of 'input()' to interact
  - Formatting output strings
## 2. Setup :
1.Pre-requisites
  - Python 3.8
  - Jupyter Notebook
2. Installation
  - installed jupyter notebook
  - used command prompt to open the jupyter notebook
