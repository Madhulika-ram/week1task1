## Personal_intro
## 1. Project Overview:
This project is a Python Jupyter Notebook program that collects a user's personal introduction and prints it in a formatted style.
## 2. Setup :
1.Install Python 3.8 or above.
2. Install Jupyter Notebook.
3. Run the notebook and enter the required details.
## 3. What i learned :
I learned how to use Python input(), format text with f-strings, work with Jupyter Notebook, and structure documentation.
